## 📌 Introduction

The C programming language is one of the most fundamental and widely used programming languages in the field of computer science. It provides a strong foundation for understanding how software works at a low level. One of the important topics covered in the C Essentials course is **string handling**, which plays a crucial role in many real-world applications.

This project, titled **“String Analyzer in C”**, is a simple yet effective program that demonstrates how strings can be processed and analyzed using basic C programming concepts. The purpose of this project is to take a string input from the user and extract meaningful information such as the number of characters, words, vowels, and consonants.

---

## 🎯 Objective of the Project

The primary objective of this project is to develop a program that can analyze a given string and provide useful insights about it. The program focuses on the following operations:

- Counting the total number of characters  
- Counting the number of words  
- Identifying and counting vowels  
- Identifying and counting consonants  

This project helps in strengthening the understanding of strings, loops, and conditional statements in C.

---

## 🧠 Concepts Used

The String Analyzer project is built using fundamental concepts from the C Essentials course:

### 1. Strings
In C, a string is represented as an array of characters terminated by a null character (`\0`). Strings are used to store and manipulate text data.

### 2. Loops
Loops such as `while` or `for` are used to traverse each character in the string.

### 3. Conditional Statements
Conditional statements like `if-else` are used to classify characters as vowels, consonants, or spaces.

### 4. Functions
Standard library functions such as `fgets()` and `tolower()` are used for input handling and case conversion.

---

## ⚙️ Working of the Program

The working of the String Analyzer program can be broken down into simple steps:

1. The program takes a string input from the user using the `fgets()` function.  
2. It initializes counters for characters, words, vowels, and consonants.  
3. A loop iterates through each character of the string.  
4. Each character is analyzed:
   - Vowels (a, e, i, o, u) are counted  
   - Alphabets other than vowels are counted as consonants  
   - Spaces are used to determine word count  
5. The process continues until the end of the string.  
6. The final results are displayed to the user.
